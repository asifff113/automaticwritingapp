Features

Everything You Need

Packed with features to make automated typing feel natural and effortless.

⌨

Natural Typing Simulation

Three typing modes - Constant, Human-like with punctuation pauses, and Burst mode with natural rhythm patterns.

⚙

Fully Configurable

Adjust speed (5-300ms), countdown (1-30s), randomness (0-100%), repeat count, and much more from the Settings tab.

💾

Preset System

Save frequently-used text as named presets. Load, overwrite, or delete them anytime. Stored locally in JSON.

⏹

Pause & Resume

Full control during typing - pause, resume, or emergency stop with F9. Never lose track of where you are.

📈

Live Progress Log

Color-coded real-time log with timestamps, progress bar, elapsed time, and ETA. Know exactly what's happening.

🎨

Beautiful Dark UI

Modern dark-themed interface with tabbed layout. Keyboard shortcuts, right-click menus, and pin-on-top toggle.

🔌

Zero Dependencies

On Windows, uses native SendInput API - no extra packages needed. Just download and run. *macOS/Linux use pynput.

💻

Cross-Platform

Works on Windows, macOS, and Linux. Pre-built binaries available for all three platforms.

📄

File I/O & Clipboard

Open .txt files, save your text, paste from clipboard. Full editor with undo/redo and context menu.
