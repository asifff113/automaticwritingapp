Features

Everything You Need

Packed with powerful features to make automated typing feel natural, efficient, and effortless.

⌨

Natural Typing Simulation

Three modes: Constant speed, Human-like with punctuation pauses, and Burst mode with natural rhythm patterns.

⚙

Speed Presets New

One-click speed presets: Slow, Normal, Fast, or Blazing. Fine-tune with the 5-300ms slider and randomness control.

🔎

Find & Replace New

Built-in Find & Replace bar with Ctrl+F. Navigate matches, replace one or all, with highlighted results.

📊

Statistics Dashboard New

Track lifetime stats: total characters typed, sessions completed, time spent. Full session history with details.

🔄

12 Text Transforms New

Uppercase, lowercase, title case, sort, reverse, deduplicate, number lines, trim, squeeze, and more.

🎨

Dark & Light Themes New

Beautiful light theme by default with a dark mode toggle. Persistent preference saved between sessions.

💾

Auto-Draft & Presets

Never lose your work. Auto-saves drafts between sessions. Named presets for frequently-used text.

⏹

Pause, Resume & Stop

Full typing control: pause, resume, or emergency stop with F9. Live progress bar, ETA, and WPM display.

📈

Live Progress Log

Color-coded real-time log with timestamps. Elapsed time, ETA, progress bar, and words-per-minute tracking.

📄

File I/O & Recent Files

Open and save text files. Quick access to recent files menu. Clipboard paste button. Full undo/redo support.

🔌

Zero Dependencies

Windows uses native SendInput API with full emoji support. No extra packages needed. *macOS/Linux use pynput.

💻

Cross-Platform

Works on Windows, macOS, and Linux. Pre-built portable binaries available for all three platforms.

💻 Windows setup instructions
  1. Download the installer (.exe) above.
  2. Double-click to run. Windows may show a SmartScreen warning — click "More info" then "Run anyway".
  3. Follow the setup wizard: Next → Accept License → Choose folder → Install.
  4. A desktop shortcut and Start Menu entry are created automatically.
  5. Launch the app from the desktop or Start Menu. To uninstall, use Add/Remove Programs.

Prefer no install? Use the portable .exe instead — just download and run, no setup needed.

🍏 macOS setup instructions
  1. Download the .zip file and double-click to extract.
  2. Right-click the extracted AutomaticWritingAssistant binary and choose "Open".
  3. macOS will ask to confirm since the app isn't from the App Store — click "Open".
  4. Grant Accessibility permission when prompted (System Settings → Privacy & Security → Accessibility).

Or from Terminal:

chmod +x AutomaticWritingAssistant && ./AutomaticWritingAssistant
🐧 Linux setup instructions
  1. Download the .zip file and extract it.
  2. Make the binary executable and run it:
chmod +x AutomaticWritingAssistant && ./AutomaticWritingAssistant

Requires a desktop session (X11 or Wayland) with input access.

Or build from source: python app.py — see instructions  |  all releases
